# Todo App

## Project Description
A simple and intuitive Todo application built using modern web technologies.

## Features
- Create, read, update, and delete tasks.
- Mark tasks as completed.
- Filter tasks based on their status.
- Responsive design for both desktop and mobile devices.

## Technologies
- HTML
- CSS
- JavaScript
- React
- Node.js

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Pedroz505/todo-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd todo-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage Guide
- Open your browser and go to `https://todo-ac586.web.app/`.
- You can start adding tasks right away.

## Configuration
- You may need to configure the server settings in `server/server.js` if you want to change the port or database settings.
